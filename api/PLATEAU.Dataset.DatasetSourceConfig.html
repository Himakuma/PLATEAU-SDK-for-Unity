    Class DatasetSourceConfig

    Parameters
    Type Name Description
    Boolean isServer

    データの場所は true ならサーバー、falseならローカルです。
    String localSourcePath

    ローカルモードでのみ利用します。インポート元のパスを渡します。
    String serverDatasetID

    サーバーモードでのみ利用します。データセットのIDを渡します。 そのIDとは、APIサーバーにデータセットの一覧を問い合わせたときに得られるID文字列です。例: 東京23区のデータセットのIDは "23ku"
    String serverUrl

    サーバーモードでのみ利用します。サーバーのURLです。
    String serverToken

    サーバーモードでのみ利用します。サーバー認証のトークンです。
