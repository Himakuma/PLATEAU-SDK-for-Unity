    Class GmlTypeTarget

    gmlファイル群は地物オブジェクトのタイプ別にフォルダ分けで格納されていますが、 そのうちのどのタイプ（フォルダ）を変換ターゲットとするかを決めます。 例: udx/bldg下のgmlは対象にするが、udx/veg下のgmlは対象にしない、など。

    他クラスとの関係は CityImporterConfig -> 保持 -> GmlSearcherConfig -> 保持 -> GmlTypeTarget という関係なので、CityImporterConfig の注意事項に基づいてこのクラスには Serializable属性が付いている必要があります。
